- Vector Gravity:
	- $$|F_g (\bold x)| = \frac {G m_1 m_2}{\bold x ^2}$$
	- $$F_g (\bold x) = \frac {G m_1 m_2}{\bold x ^2} \frac{-\bold x}{|\bold x|} = -\frac {G m_1 m_2 \bold x}{|\bold x |^3}$$
	- $$E_P(\bold x) = - \frac {G m_1 m_2}{|\bold x|}$$
	- $$E_K(\dot {\bold x}) = \frac 1 2 m \dot {\bold x}^2 = \frac 1 2 m \dot {\bold x}^2 \frac m m = \frac { m^2 \dot {\bold x}^2} {2m} = \frac {\bold p^2}{2m} = E_K(\bold p)$$
	- $$\bold s_i = \{ \bold x_i, m \dot{\bold x_i}\} = \{ \bold x_i, \bold p_i \}$$
	- $$S = \underset i \cup {\bold s_i}$$
- [A Formulation of Hamiltonian Mechanics Using Geometric Algebra](https://booksc.org/book/7257909/31094b)
	- Poisson brackets:
		- Given $$f(q_i, p_i, t)$$ and $$g(q_i, p_i, t)$$:
		  $$\{f, g\} =  \frac {\partial f}{\partial q_i} \frac {\partial g}{\partial p_i} - \frac {\partial f}{\partial p_i} \frac {\partial g}{\partial q_i} = \partial_{q_i} f \partial_{p_i} g - \partial_{p_i} f \partial_{q_i} g$$
		- $$\dot q^\mu = \partial_{p^\mu} H$$
		- $$\dot p^\mu = -\partial_{q^\mu} H$$
		- $$H = E_P(\bold q) + E_K(\bold p)$$
		- $$\nabla |\bold x|^k = k |\bold x|^{k-2}\bold x$$
		- $$\frac{d \bold p}{d t} = \partial_{\bold q} H = - \nabla G m_1 m_2 |\bold q|^{-1} = \frac{G m_1 m_2 \bold q} {|\bold q|^3} = -F_g(\bold q)$$
		- $$\frac{d \bold q}{d t} = \partial_{\bold p} H = \nabla \frac {\bold p^2} {2 m} = \frac {2 \bold p}{2m} = \frac {\bold p} m$$
		- Phase space is convenient dimensionally:
			- $$\frac{\partial H}{\partial \bold q} \equiv \frac J m = N = \frac {kg \cdot m/s} s \equiv \frac{d \bold p}{dt} = \bold{\dot{p}}$$
			- $$\frac{\partial H}{\partial \bold p} \equiv \frac J {kg \cdot m/s} = \frac m s \equiv \frac{d\bold q}{dt} = \bold {\dot q}$$
	- Configuration space: $$R^n$$
		- $$\bold q = q^\mu e_\mu$$
		- $$\partial_{\bold q} = e^\mu \partial_{q^\mu}$$
	- Momentum space: $$\bar R^n$$
		- $$\bar {\bold p} = p^\mu \bar e_\mu$$
		- $$\partial_{\bar {\bold p}} = \bar e^\mu \partial_{p_\mu}$$
	- Phase space: $$M^{2n} = R^n \bigoplus \bar R^n$$
		- [Direct sum](https://en.wikipedia.org/wiki/Direct_sum), adds two groups together into a third group containing both: $$\bigoplus$$
		- $$e_\mu \cdot \bar e_\nu = \frac 1 2 \{ e_\mu, \bar e_\nu \} = \frac 1 2 ( e_\mu \bar e_\nu + \bar e_\nu e_\mu) = 0$$
		- Define the symplectic bivector: $$J = \underset \mu \Sigma J_\mu$$
		- with component 2-blades: $$J_\mu = e_\mu \bar e_\mu = e_\mu \wedge \bar e_\mu$$
		- Such that $$e_\mu \cdot J = \bar e_\mu$$ adn $$\bar e_\mu \cdot J = - e_\mu$$
		- And also $${J_\mu}^2 = -1$$ and $$J^2 = -n$$
			- Unit imaginary: $$i = J / \sqrt n$$
		- Point in phase space: $$\theta = q + \bar p = q^\mu e_\mu + p^\mu \bar e_\mu$$
		  $$\theta = \theta^r E_r$$
			- If $$r < n$$: $$\theta^r = q^r$$, else $$\theta^r = p^{r-n}$$
			- If $$r < n$$: $$E_r = e^r$$, else $$E_r = \bar e^{r-n}$$
